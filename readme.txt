Display Live Cricket Score on your Wordpress Site.

== Description ==

Simple and Light Weight Plugin to Display Live Cricket score on your WordPress Site.

Just insert the Shortcode on Posts or pages or Widget and Done

> Make it Simple

- Shortcode

```
[hellocricket]
```

**Developer Help**

- Follow the developer [@Twitter](https://twitter.com/santhoshveerece)

== Installation ==
= Hello Cricket =
1. Go to Plugins > Add New Plugin*, search for **Hello Cricket** and click "*Install now*"
2. Alternative method, download the plugin and Upload the Hello Cricket folder to the /wp-content/plugins/directory
3. Activate the plugin
4. Now go to WordPress Dashboard -> Settings -> Plugin settings page.
5. Enter the Live Match URL and Start Displaying the Live Cricket Score via Shortcode
6. That's it
